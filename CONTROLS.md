# Tower Defense - Guide de Contrôles

## Commandes du Jeu

### Placement de Tours
- **1** : Sélectionner/Placer tour **SNIPER** (75$) - Dégâts élevés, portée longue
- **2** : Sélectionner/Placer tour **FREEZING** (50$) - Ralentit les ennemis
- **3** : Sélectionner/Placer tour **CANNON** (100$) - Dégâts AoE (zone d'effet)
- **Clic Souris** : Placer la tour au curseur (si assez d'argent)
- **ESC** : Annuler le placement de tour

### Système de Jeu
- **Objectif** : Empêcher les ennemis d'atteindre la base (tuile bleue)
- **Santé** : Commence à 20, diminue de 1 quand un ennemi atteint la base
- **Game Over** : Quand la santé ≤ 0
- **Récompense** : +10$ par ennemi tué par les tours
- **Vagues** : Augmentent progressivement (Wave 0: 3 ennemis, Wave 1: 5, etc.)

## Types de Tours

### SNIPER TOWER (Red)
- **Coût** : 75$
- **Dégâts** : 40 (élevé)
- **Portée** : 250 pixels (très long)
- **Cadence** : 0.8 tirs/sec (lent)
- **Usage** : Pour cibler des ennemis lointains avec puissance

### FREEZING TOWER (Cyan)
- **Coût** : 50$ (moins cher)
- **Dégâts** : 5 (faible)
- **Portée** : 200 pixels
- **Cadence** : 2 tirs/sec (rapide)
- **Usage** : Ralentir les ennemis pour les laisser aux autres tours

### CANNON TOWER (Yellow)
- **Coût** : 100$ (plus cher)
- **Dégâts** : 25 par projectile + AoE (zone)
- **Portée** : 180 pixels
- **Cadence** : 0.6 tirs/sec
- **Explosion Radius** : 120 pixels
- **Usage** : Dégâts multiples dans une zone, idéal pour vagues

## Stratégie Recommandée

1. **Départ** : Vous avez 300$ et 3 tours pré-placées (coût total: 225$, reste 75$)
2. **Tuer les ennemis** : Laissez les tours pré-placées combattre la vague 1 (3 ennemis)
3. **Accumulation** : 3 × 10$ = 30$ de récompense (total: 105$)
4. **Nouvelle vague** : Attendez 5 secondes avant la vague 2 (5 ennemis)
5. **Placer des tours** : Utilisez 1-3 pour ajouter de nouvelles tours stratégiquement

## Affichage à l'Écran

- **Top-Left** : Santé, vague actuelle, nombre d'ennemis
- **Top-Left (Grand)** : Argent disponible (en jaune)
- **Top-Right** : Instructions de contrôle
- **Top-Right (Sélection)** : Nom et coût de la tour en cours de placement
- **Centre (Game Over)** : Message "GAME OVER!" en rouge

## Tips

- Placez des FREEZING TOWERS pour ralentir les vagues massives
- Utilisez des SNIPER TOWERS pour éliminer les ennemis dangereux
- Les CANNON TOWERS sont idéales quand plusieurs ennemis arrivent ensemble
- Économisez l'argent pour la vague suivante si possible
- Les tours se chevauchent pour une couverture maximum
